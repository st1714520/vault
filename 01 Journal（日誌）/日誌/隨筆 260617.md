# 隨筆 260617

## 今天做的事

### 手機 vault 重建
手機上的 git 因為上次大改資料夾結構產生 merge conflict，無法解決。透過 Obsidian 的 vault 管理畫面移除舊 vault，重新 clone GitHub repo，解決了衝突問題。

### 救回遺失的筆記
重建過程中發現 vault 裡 50 個筆記檔案全部消失。追查 git 歷史後，確認是手機 commit 把新結構反轉成舊結構、再被後續 commit 全部刪除所導致。從 `ef344ab` 這個 commit 把所有檔案還原，重新 push 上 GitHub。

### 空資料夾同步問題
git 不追蹤空資料夾，手機 clone 後看不到空的子資料夾。在每個空資料夾加入 `.gitkeep` 解決。

### 下架 Quartz 公開網站
`st1714520.github.io/notes` 本來是公開筆記網站，但沒有公開筆記的需求，直接從 GitHub Pages 設定下架。

### 新增 Remotely Save 同步
git 在手機操作太麻煩，加裝 Remotely Save plugin，連接新建的 Dropbox 帳號，桌機和手機都設定好。現在日常同步用 Remotely Save，git 保留做版本歷史備份。
