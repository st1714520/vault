# 心理韌性與微笑曲線

[[危機心理支持概論]] > 心理韌性

---

## 一、微笑曲線：心理狀態的下沉與回升

危機事件的衝擊力道，決定了一個人心理狀態會下沉到哪一層。越強烈的外部壓力、越脆弱的內在資源，就越可能從正常的日常情緒層跌落至更深的病理層。

**因果邏輯**
- 原因：危機事件帶來壓力衝擊，超過個人當下的承受能力
- 結果：心理狀態依衝擊程度，沿微笑曲線向下滑落
- 關鍵變數：個人心理韌性（復原力）決定能否反彈回升

---

## 二、心理狀態四層級

| 層級 | 名稱 | 症狀 | 處置 |
|------|------|------|------|
| F1 | 日常心情（人之常情）| 短暫情緒波動 | 無需過度干預 |
| B1 | 暫時身心/行為反常 | 適應障礙、心力焦慮 | 可自行調節或短期輔助 |
| B2 | 慢性異常（精神官能症）| 焦慮症、憂鬱症、失眠症、PTSD | 需專業介入 |
| B3 | 精神病 | 妄想、幻覺、怪異言行 | 需立即醫療處置 |

---

## 三、類 PTSD 症狀三角

當個人跌入 B1–B2 層時，常出現以下三種**相互強化**的症狀：

```
驚嚇重現（Flashback）
   ↕ 相互強化
逃避忌諱  ←→  神經過敏
```

| 症狀 | 機制 |
|------|------|
| 驚嚇重現（Flashback）| 反覆出現創傷畫面 → 身體持續處於警戒狀態 |
| 逃避忌諱 | 迴避相關觸發物 → 無法正常處理創傷記憶 → 症狀難以消退 |
| 神經過敏 | 持續高度警覺 → 消耗大量身心資源 → 加重疲乏與悲哀感 |

> 三者形成惡性循環，若未介入可能惡化至 B2 層

---

## 四、心理韌性（復原力）的核心概念

心理韌性 = **允許自己暫時「掉到底部」，並相信自己具備反彈回升的力量**

**反直覺的因果推論**

| 做法 | 因果鏈 |
|------|--------|
| 壓制情緒（不允許自己掉落）| → 無法完成情緒處理 → 反而增加慢性化風險 |
| 允許情緒的正當性 | → 降低自我批判 → 反而加速回升 |
