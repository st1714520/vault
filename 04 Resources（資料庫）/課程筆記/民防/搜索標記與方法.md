# 搜索標記與方法

## FEMA 搜索標記（民間輕型搜救）

### 結構／危險標記（Structures/Hazards Marking）

標準尺寸為 2 × 2 ft（約 60 × 60 cm）的方塊。

| 標記 | 含義 |
|------|------|
| □ 空白方塊 | 結構相對安全，可直接進入執行任務 |
| ◣ 斜線方塊 | 結構明顯損壞，可能需支撐或清除危險物後才能進入 |
| ✕ 叉叉方塊 | 結構危險，不適合搜救；進入前須採取廣泛安全措施 |

方塊右側依序記錄：日期／危險種類／時間／搜救單位（TF ID）。

### 搜索評估標記（Search Assessment Marking）

| 階段 | 標記方式 |
|------|----------|
| 進入 | 畫一條斜線 `/`，左側記錄搜救單位、日期與進入時間 |
| 離開 | 補成 `✕`：上方記錄離開時間、右方記錄搜索範圍與危害、下方記錄傷亡人數 |
| 重新搜索 | 在前一標記上畫十字（`＋`）覆蓋，再重新記錄 |

常用代碼：NE（未進入）、COM（正常）、EXT（外觀評估）、RATS（鼠）、2L 1D（2 活 1 死）、O-O（無傷亡）、DOA（到場即死亡）。

> 搜索範圍如未標示，視為已搜索整個內部。若只搜索一樓，需標明（如：「搜索一樓，二樓有危險」）

**FEMA X 符號各方位含義**
- 上方：離開時間
- 左方：搜索單位／日期／進入時間
- 右方：已搜索範圍／危害因子
- 下方：倖存者狀況與傷亡人數

## INSARAG 標記系統（國際 USAR 協調）

> INSARAG 正式標記不是以「圓形＝工作區、單線＝危險區、交叉線＝禁止進入」分類。依《INSARAG Guidelines 2020, Volume II: Preparedness and Response, Manual B: Operations》，現場標記主要分為下列三類。

### 1. 工作地點標記（Worksite Marking）

- 每個工作地點由協調單位分配唯一的 **Worksite ID**，並標示於主要入口附近。
- 標記用於辨識地點及記錄隊伍作業資訊，不代表建物已被判定安全。
- 現場資訊應搭配 INSARAG Coordination & Management System（ICMS）或協調機制持續更新，不應只依賴牆面標記。

### 2. 受困者標記（Victim Marking）

| 標記        | 含義                  |
| --------- | ------------------- |
| `V`       | 疑似有受困者的位置           |
| `V` ＋方向箭頭 | 受困者方向尚未精確確定         |
| 圈住 `V`    | 已確認受困者的確切位置         |
| `L-數字`    | 已確認的生還者人數（Live）     |
| `D-數字`    | 已確認的罹難者人數（Deceased） |
| 劃除整個標記    | 該位置的受困者已全部移出        |

### 3. 快速清查標記（Rapid Clearance Marking, RCM）

| 標記 | 含義 |
|------|------|
| `C` | Clear：完成快速清查，未發現受困者或遺體 |
| `D` | Deceased only：僅發現罹難者，未發現生還者 |

RCM 應附上執行隊伍識別及日期；它只表示快速清查的結果，不等同完整搜索或結構安全認證。

> **不要混用**：封鎖帶與警戒線屬於現場管制措施，並非上述 INSARAG 搜索標記。FEMA X 標記也不可直接套用為 INSARAG 標記。

**正式來源**：[INSARAG Guidelines](https://www.insarag.org/methodology/insarag-guidelines/)（Volume II, Manual B: Operations）

## 搜索行動原則
- 最少三人行動（兩人入內、一人在外）
- Buddy System：兩人保持在手臂可觸及距離
- 呼叫語：「如果有人能聽到我的聲音，請回答我。」
- 詢問倖存者：建築物情況、其他可能受困的人
> ⚠ 遇到倖存者時，先通報，不要直接跳入救援！

## 搜索路線
**多層建築**
- 自下而上 或 自上而下
- 順時鐘 或 逆時鐘，逐層逐房清掃

**單層建築**
- 沿右牆 或 左牆前進
- 三角量測定位：多點聆聽，交叉鎖定位置

**經常停下來聆聽**：輕敲聲、移動聲、人聲

- 安全帽
- 手套
- 防塵口罩
- 護目鏡

## 搜索禁忌
- 切勿嘗試搜索有水的區域（可能有電或化學物質）
- 看到倒塌的地板或牆壁，立刻離開！
- 注意成員的體力耗損
