# 搜索標記與方法

## FEMA 搜索標記（民間輕型搜救）
| 標記 | 含義 |
|------|------|
| In / Out | 進出紀錄 |
| D / L | Dead / Live（如：1D 2L） |
| 方框內 X | 有其他小組已來過 |
| 門口方框內 X | 禁止進入 |

> 搜索範圍如未標示，視為已搜索整個內部。若只搜索一樓，需標明（如：「搜索一樓，二樓有危險」）

**FEMA X 符號各方位含義**
- 上方：搜索單位/日期時間
- 左方：進入危險
- 右方：發現情況
- 下方：倖存者/罹難者數量

---

## INSARAG 搜索標記（消防單位）
用來識別「工作區域」和「危險區域」

| 標記 | 含義 |
|------|------|
| 單條封鎖帶 | 工作區域（Operational Work Zone） |
| X 型封鎖帶 | 危險區域（Hazard Zone），禁止進入 |

---

## 搜索行動原則
- 最少三人行動（兩人入內、一人在外）
- Buddy System：兩人保持在手臂可觸及距離
- 呼叫語：「如果有人能聽到我的聲音，請回答我。」
- 詢問倖存者：建築物情況、其他可能受困的人
> ⚠ 遇到倖存者時，先通報，不要直接跳入救援！

---

## 搜索路線
**多層建築**
- 自下而上 或 自上而下
- 順時鐘 或 逆時鐘，逐層逐房清掃

**單層建築**
- 沿右牆 或 左牆前進
- 三角量測定位：多點聆聽，交叉鎖定位置

**經常停下來聆聽**：輕敲聲、移動聲、人聲

---

## 個人防護裝備
- 安全帽
- 手套
- 防塵口罩
- 護目鏡

---

## 搜索禁忌
- 切勿嘗試搜索有水的區域（可能有電或化學物質）
- 看到倒塌的地板或牆壁，立刻離開！
- 注意成員的體力耗損
