# 輕型搜救完整彙整（Light Search and Rescue）

> 本筆記以 **2024 最終簡報「輕搜技巧分享」** 為主幹，整合 [[輕型搜救概論]]、[[現場評估九步驟]]、[[建物損毀程度分級]]、[[搜索標記與方法]]、[[救援技術]] 五篇課程筆記。
> 凡簡報新增、而原筆記未收錄的內容，以 🆕 標示。

## 課程地圖
- **PART 1 現場評估** — 為什麼學、九步驟、損毀分級
- **初期應變** 🆕 — 集結、指揮、通訊、收容（原筆記未收錄）
- **PART 2 搜索與救援** — 標記系統、倒塌類型、搜索方法、疊木、搬運

## 核心原則（貫穿全課）
- **自身安全，永遠是第一要務**
- 團隊合作是搜救成功的關鍵
- **評估 → 計劃 → 執行**（持續循環，非一次性）
- 優先順序：**隊友安全 > 倖存者安全 > 環境 > 財產**

# PART 1 — 現場評估

## 為什麼學搜救
判斷是否嘗試救援取決於兩個因素：
- 救援人員面臨的**風險**
- 為最多的人達成**最大的好處**

**台灣，全球災害最密集的地方** 🆕
- **56%** 全球大型災害類型台灣都曾發生
- **1:145** 消防員與民眾比例（台灣）
- 1985 年墨西哥大地震：700 名未受訓民眾進入搜救，**100 人死在裡面**

### 黃金時機 & 誰在救人？🆕
| 角色       | 比例      | 能力                         |
| -------- | ------- | -------------------------- |
| 旁邊民眾     | **50%** | 無訓練，剛好在場直接拉出               |
| **受訓志工** | **35%** | 能執行較複雜的救援行動 ←**你就是那 35%！** |
| 輕搜隊      | 少數      | 破壞、支撐、搬運技術                 |
| 消防特搜     | 少數      | 深層搜救、工程挖掘                  |

## 四種災難類型
| # | 類型 | 主要威脅 | 核心任務 |
|---|------|----------|----------|
| 01 | 地震 Earthquake | 結構倒塌・斷層・地基液化 | 建物損毀評估 |
| 02 | 颱風 Typhoon | 洪水・土石流・斷橋・停電 | 積水環境下搜索與撤離 |
| 03 | 工傷意外 Industrial | 火災・爆炸・化學外洩・塌陷 | 優先確認危害物質種類 |
| 04 | 空襲 Air Raid | 建物破壞・火災・大規模傷亡 | 快速評估、分流、後送 |

## 輕型搜救能做什麼？🆕
| ✅ 能做 | ❌ 不能做 |
|---------|-----------|
| 移動倒塌物件與廢墟碎片 | 工程機具挖掘（重度受損建物） |
| 疊木支撐（Cribbing）建立安全空間 | 進入傾斜超標建物 |
| 進入輕度至中度受損建物執行搜救 | |
| 徒手搬運傷者至安全區域 | |

## 現場評估九步驟（9-Step Size-Up）
> 評估是**持續循環，不是一次性動作**。有效搜索取決於有效的現場評估。

### 1–3 資訊收集 ──「現場是什麼狀況？」
| # | 步驟 | 重點 |
|---|------|------|
| 1 | 收集事實 | 發生時間・建築類型・天候・瓦斯/電線/積水・受困者位置 |
| 2 | 評估損害 | 繞建物一圈觀察四個面；**裂縫 > 1mm = 明顯損壞** 🆕 |
| 3 | 考慮各種可能性 | 真的安全？還有什麼危險？斷電/火災/二次倒塌怎麼辦？ |

> ⚠ 如果不確定損毀程度，一律視為較嚴重等級！

### 4–6 評估決策 ──「我們能做什麼？先做什麼？」
| # | 步驟 | 重點 |
|---|------|------|
| 4 | 評估自身情況 | 人員・裝備・技能・心理狀態是否都準備好 |
| 5 | 確定優先順序 | 自身安全第一；先排除已知危險（瓦斯/火源/電線/阻礙物）；隊員受傷先救隊員 |
| 6 | 做出決定 | 優先救最多人・合理部署人力・確保人員精神狀態 |

### 7–9 計畫執行 ──「動起來，並持續重新評估」
| # | 步驟 | 重點 |
|---|------|------|
| 7 | 制定行動計畫 | 彙整所有資訊・文件化・動態記錄・確認指揮系統 |
| 8 | 採取行動 | 確認救援路線**及撤退路線**・防止空間迷失 |
| 9 | 持續評估 | Size-Up 循環・隨時更新現場狀況 |

## 建物損毀程度分級
> 進入建物前先判斷損毀等級。**傾斜超過高度 1/30 即達重度標準**（10m 建物 > 33cm 傾斜）🆕

| 等級 | 進入 | 特徵 | 任務 |
|------|------|------|------|
| **輕度** 🟢 | 可進入 | 窗破・表面裂縫・室內輕微損壞 | 定位倖存者、分類轉移至治療區域 |
| **中度** 🟡 | 限定進入 | 門窗框變形・明顯多處裂縫 | 立即轉移倖存者，最小化建物內人數 |
| **重度** 🔴 | 禁止進入 | 部分或全倒・傾斜・瓦斯火積水 | 任何情況不得進入！繞一圈四面觀察、通報 |

### 如何描述要去的位子
- 以建物**四個面**編號（順時鐘：道路側為一，依序二、三、四）
- 聽房子是否發出聲音、看聽危險訊號
- 判斷危險是否超出能力範圍

## PART 1 小結
> 評估 → 計畫 → 執行。分工・溝通・善用資源。**自身安全，永遠是第一要務。**

# 初期應變 🆕
> 此整段為簡報實際授課內容，原五篇筆記皆未收錄。

## 先集結，再救援
| 集結點類型 | 選擇原則 |
|-----------|----------|
| 學校操場 | 遠離可能倒塌的建築物 |
| 公園廣場 | **距離建築至少 1.5 倍樓高** |
| 空曠廣場 | 避免被周圍建物包圍；要有足夠空間容納居民 |

> **事先約定，比災後臨時決定更重要。**

## 建立簡單指揮架構
| 角色 | 職責 |
|------|------|
| 指揮 | 任務分配・人力調度・決策 |
| **通訊** | 接收各組回報・記錄資訊・傳遞消息 |
| 搜索組 | 建築物勘查・住戶確認・災情回報 |
| 救援組 | 輕度救援・引導撤離 |
| 醫療組 | 簡易急救・傷患集中管理 |

> 重點：整個系統最重要的往往不是指揮，而是「**通訊**」。

## 搜索行動原則
- **不要一出去就失聯**——出發前先約定回報時間（每 15 或 30 分鐘回報一次）
- 「30 分鐘後，不管有沒有發現，都要回來。」

**搜索的目的是**：蒐集資訊・判斷危險・回報需求
**不是**：看到人就衝進去救

## 搜索優先 & 通訊備案
| 搜索優先順序 | 通訊備案（無線電/手機失效時） |
|--------------|------------------------------|
| 完成建築評估 | 傳令員（人力傳遞） |
| 確認災情紀錄 | 手勢 |
| 確認傷患位置 | 燈號 |
| **再決定：哪裡值得投入救援** | 人力接力傳訊 |

> 停下來搶救第一個傷患，可能漏掉更多傷患與更大的危害。
> 有訊息傳遞，比使用什麼工具更重要。

## 現場判斷要點
**關係人的資訊未必可靠**（緊張・恐慌・記憶錯亂）
- 以現場觀察結果為主，不以關係人陳述作為唯一依據
- 家屬情緒激動時：安撫、提供明確資訊、阻止其衝入，避免製造更多傷患

**電的危害最難預測**：電線掉地、地面積水、設備外觀正常皆**可能仍帶電**
- → **不靠近・不接觸・立即回報**

## 遇到受困者怎麼辦
| 情況 | 處置 |
|------|------|
| 一・輕微受困（可自行或稍協助脫困）🟢 | 引導離開現場 → 帶往集結區 |
| 二・被重物壓住（超出社區隊能力）🔴 | 記錄位置 → 回報指揮站 → 呼叫專業救援 |

## 收容問題的觀念
- **社區搜救隊任務**：集結居民・控制混亂・傳遞資訊（不需建立大型收容所）
- **大規模收容由**：政府・消防單位・民政系統 負責
- 聚焦在你能做的事，讓專業單位做他們的工作。

# PART 2 — 搜索與救援

## 搜索與救援總則
- 判斷因素 1：救援人員的風險
- 判斷因素 2：救援行動的最大效益
- **安全環境三步驟：確認安全 → 搜索 → 執行救援**

## FEMA 標記系統

### 結構 / 危險標記（Structures/Hazards Marking）🆕詳解
標準尺寸 2×2 ft（60×60 cm）方塊：
| 標記 | 含義 |
|------|------|
| □ 空白方塊 | 結構相對安全，可直接進入執行任務 |
| ◣ 斜線方塊 | 結構明顯損壞，可能需支撐或清除危險物後才能進入 |
| ✕ 叉叉方塊 | 結構危險不適合搜救，進入須採取廣泛安全措施 |

> 方塊**右側**依序標記：日期 ／ 危險種類 ／ 時間 ／ 搜救單位（TF ID）

### 搜索評估標記（Search Assessment Marking）🆕詳解
| 階段 | 標記方式 |
|------|----------|
| 進入 | 劃一斜線 `/`，左側寫搜救單位與進入時間 |
| 離開 | 完成 `✕` 號：上→離開時間／右→危害或生物／下→傷亡人數 |
| 重新搜索 | 在前一標記上畫十字（＋）蓋掉，再重新記錄 |

**常用代碼**：NE(未進入)・COM(正常)・EXT(外觀評估)・RATS(鼠)・2L 1D(2活1死)・O-O(無傷亡)・DOA(到場即死亡)

### FEMA X 符號四方位（門口示意）
- 上：離開時間（Time out）
- 左：小組識別（Team ID）／日期／進入時間（Time in）
- 右：已搜索範圍（Areas Searched）／標示危害因子（Rats）
- 下：倖存者狀況（Survivor Condition）

> 搜索範圍如未標示，視為已搜索整個內部。若只搜一樓，需標明。

## INSARAG 搜索標記（消防/國際隊）
> ⚠ **與原筆記 [[搜索標記與方法]] 有出入** — 原筆記記為「單條封鎖帶=工作區域、X型=危險」；本最終簡報定義如下，以此為準。

| 標記 | 含義 |
|------|------|
| **圓形標記** | 工作區域（標示正在搜救的範圍） |
| **單條封鎖帶** | 危險區域（謹慎接近） |
| **X 型交叉封鎖帶** | 禁止進入 |

記錄格式：日期・時間・搜救隊 ID・搜索狀態（含 ASR 等級完成級別、Triage 分類、入口方向箭頭）

## 搜索準則（安全第一，進入前確認）
| 面向 | 重點 |
|------|------|
| **PPE 個人防護** | 安全帽・厚手套・口罩・護目鏡；任何現場必須穿戴完整 |
| **環境危害** | 裸露電線・瓦斯・積水・化學品；發現任一先排除或繞行 |
| **二次崩塌** | 餘震・雨水・人員重量都可能引發；隨時意識出口位置 |
| **三人制** | 最少三人（兩入一外）；任何情況不單獨進入受損建物 |

> ⚠ 切勿搜索有水的區域（可能有電或化學物質）；看到倒塌地板或牆壁立刻離開。

## 倒塌類型與結構空陷 🆕詳解
| 型態 | 英文 | 形成 | 生存率 |
|------|------|------|--------|
| 斜撐型 | Lean-to | 樓板一端倒塌、另一端固定，三角形空間 | 較高 |
| 煎餅型 | Pancake | 多層樓板逐層垂直倒塌，空間極小 | 低 |
| V 形支撐 | Supported | 樓板兩端下沉、中間支撐，V 形空間 | 中 |

**倖存者可能藏身處**：浴缸/桌下、衣櫃/冰箱/床下/床邊、電梯、車道/停車場/車內。
> ⚠ 看到倒塌地板或傾斜牆壁，立刻離開，重新評估！

## 搜索方法
| 要素 | 內容 |
|------|------|
| **3 人制** | 兩人入內搜索、一人在外保持聯絡；確認進出路線防迷失 |
| **Buddy System** | 兩人保持在手臂可觸及距離 |
| **呼叫法** | 「如果有人能聽到我的聲音，請回答我。」→ 靜音聆聽至少 30 秒再下一輪 |
| **詢問倖存者** | 建築物情況、其他可能受困的人（先通報，勿直接跳入救援） |
| **路線規劃** | 多層：由上而下逐層、順/逆時鐘逐房清掃；單層：沿右牆或左牆前進 |

### 三角量測定位法
- 三個不同位置聆聽聲音來源方向，於示意圖標記每個聆聽點方向
- 三條線交叉點 = 受困者位置
- 技巧：**敲擊管線**，聲音傳遞更快；每次呼叫後靜音聆聽 30 秒以上

## 槓桿原理與疊木支撐（Cribbing）
**用途**：移除重物、穩定兩側、幫助倖存者脫困（可就地取材）

| 角色 | 職責 |
|------|------|
| 小組長 | 指揮全程，監控整體狀況 |
| 安全手 | 持續監控結構穩定性 |
| 醫療手 | 照顧與安撫傷患 |
| 疊木手 ×2 | 放置並調整支撐材料 |
| 槓桿手 | 操作撬棒抬起重物 |

**疊木型式**：✅ 井字型（Tic-tac-toe，正確） ／ ❌ 口字型（Box，不穩定）
- 疊木間保留適當距離創造工作空間；不要把手放到重物下方
- 拖出傷者前，先把長木條放在積木側邊，防止積木被勾出

## 搬運傷者的方法（脊椎保護優先）
| 方法 | 適用 | 重點 |
|------|------|------|
| 單人攙扶 | 意識清醒・可行走 | 適合短距離撤離 |
| 單人背負 | 無法行走・意識清楚 | 距離短，注意脊椎保護 |
| 雙人肢端 | 疑似脊椎損傷 | 一人托頭頸、一人抬腳，保持脊椎平直全程固定 |
| 毯子拖拉 | 快速疏散・地面平整 | 速度快，注意保護頭部避免撞擊 |

> 原筆記 [[救援技術]] 另列：椅子搬運法、圓滾木翻身（簡報未獨立成頁，仍為可用方法）。
