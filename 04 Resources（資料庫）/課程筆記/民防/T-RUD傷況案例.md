# T-RUD 傷況案例

[[T-RUD傷患演員培訓概論]] > 傷況案例

---

## 案例 1：大出血（外出血）— 動脈破裂

**情境**：股動脈或肱動脈等大血管受損

### 症狀進展

| 時間段 | 症狀 |
|--------|------|
| 立即 | 大量噴血（動脈壓），鮮紅色 |
| 1-3 分鐘 | 頭暈、定向感喪失 |
| 4-6 分鐘 | 意識模糊、說話不連貫、肢體無力 |
| 7+ 分鐘 | 意識喪失、呼吸微弱 |

### 身體語言要點
- 用手壓住傷口（本能）
- 呼救時逐漸音量降低
- 嘗試起身但無法維持

### 對救援行動的反應

| 動作 | 反應 |
|------|------|
| 止血帶 | 強烈疼痛反應，逐漸緩和 |
| 直接加壓 | 痛苦表情，配合求生意志 |
| 放置恢復姿勢 | 意識混濁，口乾口渴 |

> **演員注意**：出血量視乎傷口位置與大血管損傷程度，噴血 vs 湧血需與導演確認

## 案例 2：內出血

**情境**：鈍傷後腹腔內臟出血（脾臟、肝臟）

### 症狀進展

| 時間段 | 症狀 |
|--------|------|
| 0-5 分鐘 | 撞擊點疼痛、噁心 |
| 5-15 分鐘 | 腹部膨脹感、血壓下降（自覺頭輕） |
| 15-30 分鐘 | 皮膚蒼白、出汗、意識混濁 |
| 30+ 分鐘 | 失去反應、血壓崩潰 |

### 外觀表現
- 外觀幾乎正常（無大型外傷）
- 腹部壓痛（防衛性肌肉緊繃）
- 姿勢上護著腹部，無法伸直

### 對救援行動的反應

| 動作 | 反應 |
|------|------|
| 觸診腹部 | 強烈拒絕、反彈防衛 |
| 建立靜脈通道 | 配合但意識持續下降 |
| 吸氧 | 輕微改善，但整體惡化 |

> **演員注意**：這是最難演的案例，外觀正常但需展現逐漸崩潰，時間軸要精確

## 案例 3：輕傷

**情境**：擦傷、小裂傷、扭傷

### 症狀
- 局部疼痛，行動輕微受限
- 意識清醒，能清楚表達傷況
- 情緒可能有輕微驚嚇或緊張

### 對救援行動的反應
- 主動配合，口頭報告傷況
- 可在協助下自行移動
- 對傷口處置（清潔、包紮）有輕微不適反應

> **演員注意**：輕傷案例常被忽視，但教導救援員不遺漏任何傷患的重要訓練

## 案例 6：全層燒燙傷（三度）

**情境**：接觸火焰、高溫液體或化學物質

### 症狀特性

| 特性 | 說明 |
|------|------|
| 無痛 | 神經末梢已被破壞，幾乎感受不到疼痛（與二度燒傷相反）|
| 外觀 | 白色或焦黑，皮革狀質地 |
| 意識 | 通常清醒，但可能進入休克 |

### 身體語言要點
- **不要表演劇烈疼痛**（三度燒傷患者通常較平靜）
- 燒傷區域不動，周邊完整皮膚仍有感覺
- 可能出現：口渴、顫抖（體溫流失）、呼吸道灼傷症狀

### 對救援行動的反應

| 動作 | 反應 |
|------|------|
| 冷水沖洗 | 僅對周邊二度燒傷部位有反應，三度區域無感覺 |
| 覆蓋保溫 | 接受，顫抖減緩 |
| 建立靜脈通道 | 皮膚堅硬，可能無法在燒傷區建立 |

> **演員注意**：三度燒傷不哭、不喊，反而是平靜但神智漸散才是真實表現

## 案例 9：氣道堵塞

**情境**：異物、腫脹或受傷造成氣道阻塞

### 症狀進展

| 時間段 | 症狀 |
|--------|------|
| 立即 | 無法說話或說話困難、喘鳴聲、恐慌 |
| 1-2 分鐘 | 用力喘氣、使用輔助呼吸肌、嘴唇發紫 |
| 3-4 分鐘 | 精疲力竭、意識改變 |
| 4-5 分鐘 | 失去意識 |

### 身體語言要點
- 本能抓喉動作
- 明顯的喘息和掙扎
- 眼神充滿恐慌，之後逐漸呆滯

### 對救援行動的反應

| 動作 | 反應 |
|------|------|
| 腹部推壓（哈姆立克）| 強烈身體配合，配合排出物道具 |
| 吸氧（面罩）| 恐慌性快速呼吸 |
| 頭頸後仰開放氣道 | 勉強配合，呼吸略有改善 |

## 案例 11：頭部外傷—意識退化（TBI）

**情境**：車禍或墜落造成的顱腦損傷

### 症狀進展（格拉斯哥昏迷指數下降）

| 時間段 | 症狀 |
|--------|------|
| 清醒期 | 短暫清醒，可能無法記住事發經過 |
| 退化期 | 定向感喪失（不知時間、地點）、語言變混亂 |
| 惡化期 | 只對疼痛刺激有反應 |
| 昏迷期 | 無反應 |

### 身體語言要點
- 瞳孔不等大（視劇本設定）
- 重複相同問題或語句（記憶形成障礙）
- 噁心、嘔吐衝動
- 可能有癲癇發作（視劇本）

### 對救援行動的反應

| 動作 | 反應 |
|------|------|
| 脊椎保護 | 配合或部分抵抗（視意識程度）|
| 語言刺激 | 早期有反應，後期漸漸無反應 |
| 疼痛刺激（評估） | 早期定向，後期僅反射 |

> **演員注意**：「清醒期」是重要教學重點，讓救援員知道外觀正常不代表無危險

## 案例 13：開放性骨折

**情境**：骨頭穿出皮膚的嚴重骨折

### 症狀
- 骨折部位可見骨骼（道具）
- 劇烈疼痛，無法移動該肢體
- 出血量依傷況設定（大腿骨折出血量大）

### 身體語言要點
- 強烈保護受傷肢體
- 對任何觸碰都有強烈疼痛反應
- 可能因疼痛休克（蒼白、出汗、心跳快）

### 對救援行動的反應

| 動作 | 反應 |
|------|------|
| 骨折固定 | 強烈疼痛，求不要動 |
| 止血（傷口周邊）| 強烈不適，但接受 |
| 搬運 | 即使小心仍有疼痛呻吟 |

> **演員注意**：重點是保護傷口道具，確保骨折道具在整個演出中保持正確位置
