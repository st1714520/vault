# 安心三部曲：助人技術

[[危機心理支持概論]] > 安心三部曲

**核心因果邏輯**：助人者自身穩定（自我調節）→ 才能透過「人我調節」感染對方 → 對方神經系統鏡像模仿 → 逐步回到容納之窗

> **角色定位**：你是「助人者」而非「拯救者」。若將自己定位為拯救者 → 承擔對方不可能全卸下的責任 → 助人者自身耗竭 → 喪失繼續助人的能力。

---

## 第一部｜二安（S & S）：建立安全基礎

**因果邏輯**：災難現場最大的心理威脅是「失控感」與「不安全感」。若無法先確保身體安全，心理安撫毫無意義。**身體安全是一切心理工作的前提。**

| 安身（Safety）| 安撫（Soothing）|
|--------------|----------------|
| 確認環境安全 → 降低外部威脅訊號 | 穩定語氣 → 降低對方交感神經激發（聲音頻率影響神經）|
| 協助避難 → 提供身體移動與主動性，對抗失控感 | 保持眼神接觸 → 活化社交神經，傳遞安全訊號 |
| 提供基礎物資 → 滿足生存需求，減少杏仁核警報 | 不帶批判的接納 → 不誘發羞恥感 → 對方敢繼續表達 |

---

## 第二部｜二解（V & V）：情緒疏通

**因果邏輯**：情緒無法被壓制，只能被流通。讓對方訴說 → 情緒外化 → 降低內在壓力鍋的張力 → 再配合感性認同，讓對方感到「被看見」→ 前額葉逐漸恢復運作。

| 舒解（Ventilation）| 理解（Validation）|
|-------------------|------------------|
| 讓對方訴說 → 情緒外化 → 降低內部壓力 | 給予感性共鳴 → 活化催產素 → 降低孤立感與防衛心 |
| 成為平穩的「迴聲板」→ 助人者穩定 → 引發對方神經鏡像模仿 | 說「難怪你會這麼怕…」→ 情緒正常化 → 降低自我批判壓力 |

---

## 第三部｜二預（P & P）：重建控制感

**因果邏輯**：失控感（無法預測未來）是災難中最強烈的心理壓力源。提供「可預測性」，能直接降低杏仁核的警報反應，讓個體重新感受到主體性。

| 預測（Prediction）| 預備（Preparation）|
|------------------|-------------------|
| 告知接下來可能的發展（如安置流程）| 規劃下一步行動，重建生活的控制感 |
| 可預測性出現 → 降低不確定焦慮 → 杏仁核警報降低 → 前額葉恢復 | 有行動方向 → 恢復主體性 → 從「受害者」轉換為「因應者」|

---

## 六、自我與人我調節

**先決條件**：助人者必須先穩定自己，才有能力穩定他人。

| 自我調節 | 人我調節 |
|---------|---------|
| 練習 SIT（壓力免疫訓練）| 用你的穩定去安撫他人的混亂 |
| 刻意暴露微壓力情境 → 反覆練習回到容納之窗 → 提高神經彈性 | 助人者保持在綠燈區 → 對方神經系統透過鏡像神經元模仿 → 逐漸被「拉入」穩定狀態 |

---

## 自我照顧四字訣：信 運 同 轉

長期心理健康不只靠技術，更靠生活方式。四個維度相互支撐，共同擴張容納之窗的寬度。

| 字 | 維度 | 因果機制 |
|----|------|---------|
| 信 | 信仰與意義 | 擁有核心意義感 → 不確定性中仍有錨定點 → 降低存在性焦慮 → 維持心理韌性 |
| 運 | 規律運動代謝 | 運動 → 代謝壓力荷爾蒙（皮質醇、腎上腺素）→ 促進神經新生 → 提高情緒基準線 |
| 同 | 同伴訴苦連結 | 社交連結 → 催產素分泌 → 降低孤立感 → 提供外在調節資源，抵禦壓力累積 |
| 轉 | 轉移注意 | 主動轉移注意 → 中斷反芻思考 → 讓神經系統暫時休息 → 下次面對壓力時更有餘力 |
