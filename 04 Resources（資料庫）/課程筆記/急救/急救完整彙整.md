# 急救完整彙整

本筆記整理一般急救、基本救命術、常見創傷、災難急救站及大量傷病患應變。核心目標不是在現場完成診斷，而是：**確保安全、辨識立即生命威脅、呼叫支援、提供能力範圍內的處置並安全交班。**

## 通用急救流程

1. **確認安全**：交通、火災、電力、瓦斯、積水、倒塌、暴力及感染風險。
2. **使用防護**：手套；有飛濺風險時使用口罩與護目鏡。
3. **確認反應與呼吸**：呼叫並輕拍肩膀，觀察是否有正常呼吸，時間不超過 10 秒。
4. **呼叫支援**：撥打 119、開擴音，請旁人取得 AED 或急救包。
5. **處理立即生命威脅**：無正常呼吸先 CPR／AED；大量出血先止血。
6. **持續監測**：意識、呼吸、膚色及病情變化，避免低體溫。
7. **交班**：說明事件經過、發現的問題、生命徵象與已做處置。

> 不安全就不進入。若現場持續有威脅，先通報消防、警察或相關專業單位。

# 基本救命術：成人 CPR 與 AED

## 何時開始

發現成人**無反應且沒有正常呼吸**（包含只有瀕死式喘息）時：

1. 撥打 119 並請人取得 AED。
2. 立即開始胸外按壓。
3. AED 到達後立即開機並依語音操作。

## 高品質胸外按壓

- 位置：胸口中央。
- 成人深度：約 5–6 公分。
- 速率：每分鐘 100–120 次。
- 每次讓胸廓完全回彈。
- 除 AED 分析、環境危險或交由專業人員接手外，盡量不中斷。

若受過人工呼吸訓練且願意施作，可採 30 次按壓配 2 次吹氣；不熟悉時持續徒手按壓並遵從 119 指示。

## AED

1. 開機。
2. 依圖示貼妥貼片；胸部潮濕時先快速擦乾。
3. AED 分析時所有人離開傷患。
4. 建議電擊時確認無人接觸，再按下電擊鍵。
5. 電擊後或不建議電擊時，立即恢復 CPR。

# 兒童與嬰兒

兒童與嬰兒的心肺停止常與呼吸問題有關，因此人工呼吸更重要。按壓深度約為胸廓前後徑三分之一：兒童約 5 公分、嬰兒約 4 公分，速率仍為每分鐘 100–120 次。

- 兒童可依體型使用單手或雙手按壓胸口中央。
- 嬰兒單人急救可用兩指按壓胸口中央、兩乳連線稍下方。
- AED 優先使用小兒模式或小兒貼片；沒有時依 119 指示使用可取得的 AED，不要延誤急救。

兒童與嬰兒急救強烈建議完成正式 BLS 實作課程。

# 異物梗塞

## 一歲以上兒童與成人

- 能咳嗽或發聲：鼓勵持續用力咳嗽並密切觀察。
- 無法咳嗽、說話或呼吸：撥打 119，依現行訓練執行背擊與腹部推擠。
- 失去意識：安全放倒，開始 CPR；每次開放呼吸道時只取出清楚可見的異物，不盲目挖取。

孕婦或明顯肥胖者使用胸部推擠，並依 119 指導。

## 一歲以下嬰兒

採 5 次背擊與 5 次胸部推擠交替進行。不要對嬰兒施作腹部推擠。若失去意識，開始嬰兒 CPR 並配合 119 指示。

# 大出血與休克

## 辨識致命性出血

- 血液持續大量流出或噴射。
- 衣物或敷料快速浸透。
- 地面形成明顯血泊。
- 創傷性截肢或深部傷口。

## 一般民眾處置

1. 戴手套並撥打 119。
2. 以紗布或乾淨布料直接、持續加壓。
3. 血液滲出時在原敷料上加疊，不反覆掀開。
4. 接受過訓練且直接加壓無效時，依訓練使用傷口填塞或止血帶。
5. 記錄止血帶時間；不要自行鬆開。
6. 讓傷患休息、保暖並持續觀察意識與呼吸。

休克傷患可能蒼白濕冷、焦躁、口渴、呼吸與脈搏加快或意識下降。不要給予飲食，儘速後送。

# 常見創傷處置

## 穿刺物

- 不要拔除。
- 以敷料固定物體周圍，避免晃動。
- 立即撥打 119。

## 骨折與扭傷

- 不強行復位。
- 以發現姿勢支持與固定。
- 固定前後觀察末端膚色、溫度、感覺及動作。
- 開放性骨折以乾淨敷料覆蓋，不把外露骨頭推回。

## 頭頸與脊椎傷害

- 避免不必要移動及頭頸旋轉。
- 若環境安全，維持原位並持續觀察。
- 意識下降、反覆嘔吐、抽搐、瞳孔不等大或耳鼻流液均屬緊急警訊。
- 呼吸道與立即生命威脅仍優先處理。

## 燒燙傷

1. 先停止燃燒來源並確認施救者安全。
2. 以流動的涼水降溫；不要使用冰塊、牙膏、油脂或不明藥膏。
3. 移除未黏住的飾品與緊身物；不要撕除黏住的衣物。
4. 以乾淨、不黏傷口的敷料鬆鬆覆蓋並避免失溫。
5. 大面積、深層、臉頸、手足、會陰、環狀、電擊或化學燒傷應立即求援。

化學灼傷先依物質安全指引去除污染並大量沖洗；粉末應先避免擴散並依專業指示處理。電擊傷先斷電，未確認斷電前不可接觸傷患。

# 非創傷急症的快速辨識

## 中風 FAST

- F — Face：臉歪。
- A — Arm：單側手臂無力。
- S — Speech：說話不清或無法理解。
- T — Time：記錄最後正常時間，立即撥打 119。

不要自行給予食物、飲水或藥物。

## 疑似心肌梗塞

胸悶胸痛、冒冷汗、噁心、呼吸困難，或疼痛延伸至手臂、背部、下巴時：停止活動、採舒適姿勢並撥打 119。若失去反應且無正常呼吸，立即 CPR／AED。

## 癲癇發作

- 移開周圍危險物並保護頭部。
- 不壓制肢體、不塞東西進嘴巴。
- 記錄發作時間。
- 發作後確認呼吸並陪伴恢復。

首次發作、超過 5 分鐘、反覆發作、受傷、懷孕或水中發作時立即撥打 119。

# MARCH 與一般急救的關係

一般急救先判斷反應與呼吸，適合日常突發急症；MARCH 則針對高危環境及嚴重創傷，把致命性大出血放在最前面。

| 情境 | 優先架構 |
|------|----------|
| 一般成人突然倒下 | 反應／呼吸 → 119 → CPR／AED |
| 嚴重外傷且大量出血 | 安全 → MARCH，先控制大出血 |
| 大量傷病患 | CSCATTT 建立管理與分流，再依個別傷患使用 MARCH 等臨床架構 |

詳細流程見 [[MARCH-PAWS完整彙整]]。

# 災難急救站

急救站不是只有一張治療桌，而是一套能持續運作的災難醫療系統。規劃時可用 **4S**：

| 面向 | 內容 |
|------|------|
| Staff | 有效人力、角色、排班、培訓及留任 |
| Stuff | 醫藥衛材、PPE、通訊、照明、電力、水及後勤物資 |
| Space & Structure | 檢傷、候診、治療、留觀、清消、轉診與人員休息空間 |
| System | 啟動標準、指揮、紀錄、感染控制、供應鏈、轉診與備援流程 |

## CSCATTT 大量傷病患架構

### 醫療管理

- C — Command：指揮管理。
- S — Safety：安全。
- C — Communication：溝通協調。
- A — Assessment：整體評估。

### 醫療作業

- T — Triage：檢傷分類。
- T — Treatment：治療。
- T — Transport：後送與雙向轉診。

急救站應維持單向且清楚的病患動線，分開檢傷、候診、處置、留觀、出院與後送區域，同時規劃隱私、清消、手部衛生、廢棄物、備援電力及工作人員休息。

# 交班與紀錄

使用 MIST：

- M：受傷機轉或主訴。
- I：已發現的傷勢或疾病。
- S：症狀、意識與生命徵象及其趨勢。
- T：已完成處置、時間及反應。

至少記錄：事件時間、最初發現、傷患身分、過敏與病史（若可得）、評估結果、處置內容、止血帶時間及交接對象。
