SCOPE / 輕型搜救

做能力所及的事，
不進入能力之外。

CAN DO

能做

  • 現場評估與風險辨識
  • 表面搜索與呼叫
  • 簡易槓桿與穩固
  • 安全搬運與資訊回報
STOP / HAND OVER

  • 進入重度損毀結構
  • 無防護處理瓦斯、電力
  • 單獨行動或越級操作
  • 讓救援者成為下一位傷患

PART 01 / SCENE SIZE-UP

現場評估
九步驟

不是清單，是持續循環。
01

收集事實

時間、地點、事件、受影響範圍。

02

評估損害

梁柱、樓板、裂縫、傾斜與落物。

03

考慮可能性

還會發生什麼？次生風險在哪裡？

04

評估能力

人力、裝備、訓練與限制。

05

排定優先

先處理最危急且能介入的事。

06

做出決定

進入、等待、撤退或請求支援。

07

制定計畫

任務、角色、路線與撤離點。

08

採取行動

明確溝通，依計畫執行。

09

持續評估

環境一變，重新回到第一步。

DAMAGE / 損毀程度分級

看懂結構，
才知道該前進或退出。

LIGHT

輕度

可控

表面裂縫、小範圍非結構損壞；仍需持續觀察。

MODERATE

中度

警戒

明顯裂縫、局部變形；限制接近並請專業評估。

HEAVY

重度

撤離

結構傾斜達 1/30、樓板倒塌或主要構件破壞。

優先順序隊友安全›倖存者›環境›財產

MARKING / 留下可讀資訊

你離開後，
標記繼續說話。

搜索標記讓後續隊伍知道誰來過、何時搜索、發現什麼，以及現場仍有哪些危害。

FEMA SEARCH MARK

02

電線、瓦斯、積水與火源。

Buddy System

互相監看、保持視線或語音聯絡。

VOID / 倒塌與空陷

空間可能
保存生命，
也可能再次坍塌。

辨識斜靠、V 型與層疊空陷。看到倒塌樓板、持續位移或異常聲響，立刻退出並交由專業隊伍。

看到倒塌樓板 → 立刻離開
LOCATE / 三角量測定位
ABC?

安靜、呼叫、
聆聽、定位。

最少三人從不同位置聆聽，以聲音方向交會縮小範圍。逐區記錄，不用猜測取代確認。

「有人聽得到嗎？請敲擊三下！」
RESCUE / 槓桿與穩固

先穩定，
再抬升。

槓桿負責創造空間，疊木負責保住空間。每次小幅抬升、同步填入支撐，任何人都不把手腳伸入未穩固區域。

LOAD支點

井字型

穩定、可逐層增高。

口字型

依荷重與材料配置。

MOVE / 搬運傷者

選擇最安全、
最少移動的方法。

01

單人攙扶

清醒且能部分負重。

02

單人背負

短距離且救援者能力足夠。

03

雙人肢端

協調口令，同步抬起。

04

椅子搬運

狹窄空間與樓梯需格外謹慎。

05

毯子拖拉

低姿勢撤離，保護頭頸。
