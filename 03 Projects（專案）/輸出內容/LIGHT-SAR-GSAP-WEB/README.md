# 輕型搜救訓練課程｜GSAP 網站

內容依 `輕型搜救訓練課程-簡報架構.md` 改編為長頁互動網站，分為現場評估、搜索與救援兩大段。

直接開啟 `index.html` 即可預覽。使用 GSAP 3.13 與 ScrollTrigger CDN，含首頁時間軸、九步驟橫向任務流、滾動進度、圖片視差與響應式／減少動態效果支援。

網站使用三張專案內 AI 生成插圖：

- `assets/rescue-hero.png`：封面搜救志工
- `assets/buddy-search.png`：伙伴系統搜索場景
- `assets/collapse-voids.png`：倒塌空陷型態

生成方式：Codex 內建 Image Generation。風格統一為年度報告式編輯插畫，白／天藍／金黃／黑色系，無文字與品牌標誌。
