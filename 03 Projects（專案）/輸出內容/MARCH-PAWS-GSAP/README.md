- `04 Resources（資料庫）/課程筆記/急救/MARCH-PAWS完整彙整.md`

## 預覽

直接開啟 `index.html`，或以 Node.js 執行 `node serve.mjs` 後前往 `http://127.0.0.1:4173`。

動畫使用 GSAP 3.13 與 ScrollTrigger CDN。頁面包含首頁時間軸、滾動進度、分段進場、循環重評動畫、重播按鈕、響應式版面與 `prefers-reduced-motion` 支援。

> 本頁為學習摘要，不取代正式訓練、醫療指示或 119 派遣員指導。
