# 危機情境中的心理支持

Psychological Support in Crisis Situations

## 核心邏輯

```
災難/危機事件
  → 造成心理失衡
  → 理解心理反應機制
  → 擴展自我調節能力
  → 掌握安心助人技術
  → 協助他人回到穩定狀態
```

## 整體因果架構

```
危機事件衝擊
  ↓
神經系統失調（紅燈/藍燈區）+ 類PTSD症狀三角形成
  ↓
理解機制（容納之窗、微笑曲線）→ 降低自我批判，允許反應正當化
  ↓
急性調節（回到容納之窗的策略）→ 恢復即時功能
  ↓
長期建設（擴張容納之窗）→ 提高壓力承受上限
  ↓
助人三部曲（S&S → V&V → P&P）→ 協助他人恢復穩定
  ↓
自我照顧四字訣（信運同轉）→ 維持助人者的長期韌性
```

## 與民防訓練的關係

| 民防場景 | 對應心理概念 |
|---------|------------|
| 災難現場協調 | 容納之窗——保持自身在綠燈區 |
| 協助傷患 | 安心三部曲（S&S → V&V → P&P） |
| 高壓下決策 | 自我調節 → 前額葉維持運作 |
| 長期志工/救援工作 | 自我照顧四字訣，防止耗竭 |
| 傷患演員訓練（T-RUD）| 下角色 = 心理韌性、感官錨定 = 正念技術 |
